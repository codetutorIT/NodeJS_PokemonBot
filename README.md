# NodeJS_PokemonBot

NodeJS bot for Mudae Pokemon Bot on Discord

## Prerequisites

```
npm install node-telegram-bot-api
npm install discord.js
npm install node-schedule
```

## Deployment

```
node pkmbot.js &
```
